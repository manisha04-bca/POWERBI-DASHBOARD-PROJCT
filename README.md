# Power BI Student Dashboard

## Project Overview
This Power BI dashboard analyzes student performance using:
- Exam Scores
- Attendance
- Gender
- Parental Involvement
- Motivation Level

## Features
- Interactive dashboard with slicers
- Visual comparison of exam scores
- Attendance and performance insights
- Filters for better analysis

## Tools Used
- Power BI Desktop
- Microsoft Excel
- GitHub

## How to Use
1. Download the .pbix file
2. Open using Power BI Desktop
3. Interact with slicers to explore insights
